## About the Project

This Capstone project aims to evaluate the business processes and performance of the Northwind company by analyzing various datasets. Analyses have been conducted using PostgreSQL, with some visualized using Python and PowerBI.

## Data Tables

The data analyzed in this project consists of the following tables:

1. **categories**
2. **products**
3. **suppliers**
4. **order_details**
5. **orders**
6. **customers**
7. **customer_customer_demo**
8. **customer_demographics**
9. **employees**
10. **employee_territories**
11. **territories**
12. **region**
13. **shippers**
14. **us_states**

## Analyses

### Employee Analysis
Analyses evaluating employee performances and demographic information, visualized using PowerBI.

### Financial Analysis
Analyses assessing the company's financial status and performance, visualized using PowerBI.

### Logistics Analysis
Analyses evaluating shipping and logistics processes, visualized using PostgreSQL queries as well as PowerBI and Python.

### Customer Analysis
Analyses assessing customer behaviors and demographic information, visualized using PowerBI.

### Sales Analysis
Analyses evaluating sales performance, visualized using PowerBI.

### Inventory Analysis
Analyses assessing inventory status and management, visualized using both PostgreSQL queries and Python.

### RFM Analysis
RFM analysis evaluating customer segmentation.

## Files

This repository contains the following files:

- **EmrahBas_PowerBI_Capstone.pbix**: PowerBI file containing various visualizations of the analyses.
- **EmrahBas_PythonCapstone.ipynb**: Python Jupyter Notebook file containing visualizations of inventory and logistics analyses.
- **EmrahBas_SQL_Capstone.pdf**: PDF file containing PostgreSQL queries.
- **NorthWind.png**: PowerBI screenshot of the Northwind company.
- **Northwind_Overview.png**: PowerBI screenshot providing an overview of the Northwind company.
- **NorthWindERD.png**: ERD visual of the Northwind data.

## Usage

1. **PowerBI File**: You can open the `EmrahBas_PowerBI_Capstone.pbix` file with PowerBI Desktop to view the analyses.
2. **Python Notebook**: Open the `EmrahBas_PythonCapstone.ipynb` file with Jupyter Notebook or Jupyter Lab to review the visualizations made with Python.
3. **PostgreSQL Queries**: Review the PostgreSQL queries by examining the `EmrahBas_SQL_Capstone.pdf` file.
